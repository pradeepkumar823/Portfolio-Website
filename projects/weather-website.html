Project Overview

A responsive weather information website that provides real-time weather data for any location worldwide. Built with vanilla JavaScript and integrated with weather APIs for accurate and up-to-date information.

Features a clean, intuitive interface with dynamic weather icons, temperature displays, and detailed weather conditions. Implements geolocation for automatic local weather detection and search functionality for any city globally.
