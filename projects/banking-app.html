Project Overview

A full-stack banking application that simulates core banking operations with enterprise-grade security. The system implements BCrypt password hashing, Spring Security for authentication, and comprehensive transaction management.

Features include real-time transaction logging, deposit and withdrawal operations, account balance management, and complete transaction history with timestamps and descriptions.
