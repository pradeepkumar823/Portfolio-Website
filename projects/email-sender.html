Project Overview

A full-stack email application that enables users to send, manage, and track emails through an intuitive React frontend and robust Spring Boot backend. The application integrates with SMTP servers for reliable email delivery.

Features a modern React UI with form validation, real-time status updates, and a RESTful backend API built with Spring Boot. Implements JavaMail API for email operations with support for attachments and HTML content.
