Project Overview

GrievanceHub is a comprehensive citizen complaint management system designed to streamline the process of filing, tracking, and resolving public grievances. The platform implements dual authentication mechanisms (OAuth2 + JWT) for enhanced security.

The system features a normalized database schema that reduces data redundancy by 30%, ensuring efficient data storage and retrieval. Role-based access control ensures that citizens, administrators, and officials have appropriate access levels.
