Project Overview

A faithful recreation of the popular 2048 puzzle game with smooth animations, score tracking, and responsive controls. Built entirely with vanilla JavaScript for optimal performance and learning experience.

Implements game logic including tile merging, movement algorithms, win/loss detection, and score calculation. Features local storage for high score persistence and keyboard/touch controls for accessibility.

Classic Gameplay

Score Tracking

Arrow key support for desktop gameplay with responsive key handling.

Touch Support

Swipe gestures for mobile devices ensuring cross-platform compatibility.

New Game

Quick restart functionality to begin new games without page reload.
